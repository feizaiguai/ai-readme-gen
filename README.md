# AI Readme Generator

智能 README 生成器，自动分析项目结构和代码生成专业的 README 文档。

## 功能特性

- 📊 自动分析项目结构
- 🌐 支持多语言项目（Python/JavaScript/Go/Rust 等）
- 📝 生成完整的目录结构
- 🔧 自动检测依赖并生成安装指南
- 📋 生成贡献指南和许可证

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
# 生成项目的 README
python main.py ./my-project

# 生成当前目录的 README
python main.py .
```

## 生成内容

生成的 README 包含：
- 📖 项目简介
- 📁 目录结构
- 🚀 安装指南
- 💡 使用方法
- 🤝 贡献指南
- 📄 许可证
- 📧 联系方式

## 支持的语言

- Python (.py)
- JavaScript (.js)
- TypeScript (.ts)
- React (.jsx, .tsx)
- Java (.java)
- Go (.go)
- Rust (.rs)
- C/C++ (.c, .cpp)
